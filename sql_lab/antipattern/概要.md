# SQLアンチパターン学習

## 構築

```bash
# コンテナを起動（1_Intro/setup.sqlが動くようになっている）
docker compose up -d
# データベースに接続
psql -h localhost -p 5435 -U test-user -d anti_db
```

## 演習

各章でDDLを実行することが多いため、`./sql_lab/antipattern/1_Intro/setup.sql` は参考用DDLである。
`./sql_lab/antipattern/` 配下に章ごとにフォルダを作り、その配下にSQLファイルを用意しDDLやDMLを作成し実行する必要がある。
