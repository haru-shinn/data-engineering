## ロジカルレプリケーション（テーブル単位）

### 準備（WSL2(ホスト)）

```bash
# コンテナの起動
docker compose up -d primary_logical secondary_logical

# コンテナの状態確認
docker container ps
docker logs postgres_primary_logical
docker logs postgres_secondary_logical

# データベースへの接続
psql -h localhost -p 5435 -U test_user -d primary_db
psql -h localhost -p 5436 -U test_user -d secondary_db

```

### プライマリデータベースの準備と公開設定

```bash
# 送信対象のテーブルの作成
primary_db=# CREATE TABLE users (id int PRIMARY KEY, name TEXT);
primary_db=# INSERT INTO users VALUES (1, 'abcde');
# 送信対象外のテーブルの作成
primary_db=# CREATE TABLE stores (id int PRIMARY KEY, name TEXT);
primary_db=# INSERT INTO stores VALUES (101, 'shinjuku');
primary_db=# \dt
          List of relations
 Schema |  Name  | Type  |   Owner
--------+--------+-------+-----------
 public | stores | table | test_user
 public | users  | table | test_user
(2 rows)

# 公開設定
primary_db=# CREATE PUBLICATION my_pub FOR TABLE users;
```

### セカンダリデータベースの準備と受信設定

```bash
# 受信対象のテーブル作成
secondary_db=# CREATE TABLE users (id int PRIMARY KEY, name TEXT);

# 受信設定
secondary_db=# CREATE SUBSCRIPTION my_sub
CONNECTION 'host=postgres_primary_logical port=5432 user=test_user password=password dbname=primary_db'
PUBLICATION my_pub;
NOTICE:  created replication slot "my_sub" on publisher
CREATE SUBSCRIPTION

secondary_db-# \dt
         List of relations
 Schema | Name  | Type  |   Owner
--------+-------+-------+-----------
 public | users | table | test_user
(1 row)

primary_db=# SELECT COUNT(*) FROM users;

-------
     1
(1 row)

```

## ロジカルレプリケーション（スキーマ単位）

- 事前準備は「ロジカルレプリケーション（テーブル単位）」を参照。

### 送信側でスキーマ作成と公開設定

```bash
# プライマリデータベース
primary_db=# CREATE SCHEMA sales;
primary_db=# CREATE TABLE sales.orders (id int PRIMARY KEY, amount int);
primary_db=# CREATE PUBLICATION sales_pub FOR TABLES IN SCHEMA sales;
```

### 受信側で受信設定

```bash
# セカンダリデータベース
secondary_db=# CREATE SCHEMA sales;
secondary_db=# CREATE TABLE sales.orders (id int PRIMARY KEY, amount int);

secondary_db=# CREATE SUBSCRIPTION sales_sub
CONNECTION 'host=postgres_primary_logical port=5432 user=test_user password=password dbname=primary_db'
PUBLICATION sales_pub;
NOTICE:  created replication slot "sales_sub" on publisher
CREATE SUBSCRIPTION
```

### 新しいテーブルの作成

```bash
# プライマリデータベース
primary_db=# CREATE TABLE sales.items (id int PRIMARY KEY, name text);
primary_db=# INSERT INTO sales.items VALUES (1, 'Laptop');

# セカンダリデータベース
secondary_db=# CREATE TABLE sales.items (id int PRIMARY KEY, name text);
secondary_db=# ALTER SUBSCRIPTION sales_sub REFRESH PUBLICATION;
secondary_db=# \dt sales.*
          List of relations

--------+--------+-------+-----------

(2 rows)
```
